# Flutter Horizontal Scroll Data Table Sample

A compact Flutter demo for developers searching for a scrollable data table sample with text selection, mouse drag scrolling, and expandable rows.

## Why this project

This sample is ideal for anyone building:

- desktop or web dashboards with wide tables
- horizontally scrollable data grids
- apps that require text selection across interactive rows
- expandable detail rows inside a long list

## Highlights

- `SelectionArea` support for selecting text anywhere in the table
- Custom `MouseDraggableScrollBehavior` for a better desktop/web scroll experience
- Horizontal `SingleChildScrollView` with a fixed table width
- Expandable rows rendered with `AnimatedSize`
- Snack bar simulation for inline document link interaction
- Simple Material 3 styling with responsive row layout

## Getting Started

1. Open the project in VS Code, Android Studio, or any terminal.
2. Install dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```

4. To run on a specific device:

```bash
flutter run -d <device_id>
```

## Key files

- `lib/main.dart` — main UI and sample table implementation
- `pubspec.yaml` — Flutter package configuration

## Project info

- Flutter SDK: compatible with `sdk: ^3.11.5`
- Supports mobile, web, and desktop platforms
- Sample is configured for local development only (`publish_to: none`)

## Notes

- This project is best used as a learning or prototyping sample.
